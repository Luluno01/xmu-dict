# Setup

## Install sails.js

```@Shell
npm install sails -g
```

## Clone

```@Shell
cd some/working/directory/
git clone https://github.com/Luluno01/xmu-dict.git
```

## Install Dependencies

```@Shell
cd some/working/directory/xmu-dict/
npm install
```

## Run Server

```@Shell
sails lift
```
